# 🤖 UI Automation con Selenium & Python (POM)

Este proyecto automatiza un flujo de E-commerce completo utilizando el patrón de diseño **Page Object Model (POM)**.

## 🚀 Características
- **Framework:** Pytest
- **Herramienta:** Selenium WebDriver
- **Patrón de Diseño:** Page Object Model para alta mantenibilidad.
- **Auto-gestión de Drivers:** Uso de WebDriver Manager.

## 🛒 Escenario Automatizado
- Login con credenciales válidas.
- Selección de productos dinámicos.
- Gestión del carrito de compras.
- Validación de integridad del checkout.
